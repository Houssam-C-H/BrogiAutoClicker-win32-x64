How to use
1
Configure: Set your interval, click type, and location mode in the Main tab.
2
Start Clicking: Press F6 (default) or click the large START button. The app will pulse green while active.
3
Pick Location: If using 'Fixed Position', press F7. The app will minimize, giving you 3 seconds to move your mouse to the target spot.
Pro Tips
  • Background Mode: You can minimize the app or play games; the hotkeys will still work!
  • Autosave: Every change you make is saved instantly. No need to re-enter settings next time.
  • Low Latency: For maximum speed, set interval to 10ms.

Brogi Auto Clicker is designed to be lightweight and zero-dependency. It uses Windows PowerShell for high-precision clicking.
